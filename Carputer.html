Project Overview

For my senior project, I reassembled a functional Dell computer within the shell of a hot rod mailbox. This unique project was mandatory for high school graduation and showcased creativity, technical skills, and problem-solving. The project emphasized merging aesthetic design with functional hardware.

Key Goals and Objectives

Goal:

  • Create a custom-built computer housed in a creative, unconventional case.

Objectives:

  • Assemble the computer hardware within the hot rod mailbox.
  • Ensure the system boots and runs Windows 10 OS efficiently.
  • Address any technical and design challenges encountered.

The project focused on hardware assembly, aesthetic modification and functionality testing. It includes planning, case modification, component integration and operating system installation.

Technologies Used

Hardware:

  • Standard PC components (motherboard, CPU, RAM, SSD, PSU).

Software:

  • Windows 10 OS.

Tools:

  • Screwdrivers, soldering iron, dremel tool for case modification.

Challenges & Solutions

Challenge: Issues with the system booting and BIOS errors.

  • Solution: I updated the BIOS firmware to ensure compatability with Windows 10 and reseated all components to ensure proper connections.

Challenge: Designing airflow in the unconventional case.

  • Solution: Drilled strategic ventilation holes and added small fans for heat dissipation.

The final product was a functional, visually unique computer demonstrating innovation and engineering skills.
